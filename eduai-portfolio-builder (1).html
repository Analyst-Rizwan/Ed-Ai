Deploy to GitHub Pages
Your portfolio will be live at https://mohamad-dev.github.io
✅ This generates a complete repository with a GitHub Actions workflow that automatically builds and deploys your portfolio every time you push a change. Free forever on GitHub Pages.
1
Download your portfolio files
Click below to download a ZIP containing your index.html and the GitHub Actions deploy workflow.
2
Create a GitHub repository
Go to GitHub and create a new repository. For the portfolio to appear at username.github.io, name it exactly:
mohamad-dev.github.io
3
Push your files
Open a terminal, unzip the downloaded folder, and run:
git init git add . git commit -m "Initial portfolio deploy" git branch -M main git remote add origin https://github.com/mohamad-dev/mohamad-dev.github.io.git git push -u origin main
4
Enable GitHub Pages
In your repo: Settings → Pages → Source → GitHub Actions. The workflow in .github/workflows/deploy.yml will automatically trigger on every push and deploy your portfolio. Usually takes 2–3 minutes.
5
Auto-deploy on changes
Every time you update your portfolio in EduAI and download a new version, just run git add . && git commit -m "Update portfolio" && git push and GitHub Actions will redeploy automatically. No manual steps.
